# Multi-Utility Student Dashboard 
The project consists of a dashboard containing various features for a student:<br />
-Creating Notes<br />
-Listing Homework<br />
-Youtube Search<br />
-To-Do list<br />
-E-Books<br />
-Dictionary<br />
-Wikipedia<br />
-Unit Conversion<br />
It also contains user authentication by registration and login features
